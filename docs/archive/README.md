# Docs Archive

This folder contains historical handoffs, superseded plans, and old spec packs.
It is not a source of truth for current behavior.

Completed implementation/follow-up plans such as the realtime voice, media
attachments, and Google Calendar plans also live here once `docs/current-state.md`
has absorbed the implemented behavior and remaining hardening notes.

Before using anything here, read `docs/current-state.md` and inspect the current
code. Code wins when archived Markdown disagrees with implemented behavior.
